# Docmark

[![NPM version][npm-img]][npm-url]
[![License][license-img]][license-url]
[![Build status][travis-img]][travis-url]

Extract Markdown content from doc comments.

## Installation

~~~ text
npm install docmark
~~~

## Usage

~~~ text
docmark <input> <output>
~~~
